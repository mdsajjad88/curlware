******Instructions on how to run the application.*****
Step-1: At First Clone this project.
Step-2: composer install/update
Step-3: mysql database create(curlware)
Step-4: Key Generate(php artisan key:generate)
Step-5: env file update (php artisan config:cache or php artisan optimize)
Step-6: php artisan migrate
Step-7: Run this project 
Step-8: Create new task and check all operation (operation Controller = NewtaskController)
*******END**********
